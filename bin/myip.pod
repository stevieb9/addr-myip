myip - Get your public IPv4 or IPv6 address

=head1 DESCRIPTION

Command line utility to get and display your public IPv4 or IPv6 address.

=head1 USAGE

    # Display IPv4 address

    myip

    # Display IPv6 address (if available)

    myip 6
